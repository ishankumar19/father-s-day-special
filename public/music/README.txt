Drop a single mp3 file here named:

  background-music.mp3

This is already wired up in src/data/content.js (see the "music" key) and
played via the floating note button in the bottom-right corner of the
site. If the file isn't present, that button simply disables itself —
nothing breaks.

Keep the file reasonably small (a 2–4 minute, compressed mp3 is plenty)
so the site stays quick to load. Make sure you have the rights to use
whatever track you choose.
